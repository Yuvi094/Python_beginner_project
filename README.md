1. Guessing game:
  * Features of this game 
      -This is a number guessing game I made using Python.
      -It was organized to be a fun challenge for anyone who wants to test their luck.
      -There are 5 difficulty levels to choose from:
          -Easy (1-50)
          -Medium (1-100)
          -Hard (1-500)
          -Extreme (1-1000)
          -Impossible (1-10000)
  * How to play this and a conclusion
      -First, the player enters their name and then picks how hard they want the game to be.
      -If you guess wrong, the program asks if you want a hint.
      -The hint tells you to go higher or lower, which is very much in need when the range is large.
      -I used a try and except block so the code is successful even if someone types something wrong.
      -Overall, this project helps with learning how loops and logic work together in Python.
  
  * How to use this:
      -Make sure Python is installed on your computer.
      -Run the file in your terminal usinf python Guessing_game.py command.
      -Follow the steps to find the lucky number.
